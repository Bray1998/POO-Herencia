# Proyecto POO-Herencia

Este proyecto demuestra el uso de **herencia en Java**.

## Archivos
1. Persona.java → Clase base con atributos nombre y edad.
2. Estudiante.java → Subclase que hereda de Persona y agrega carrera.
3. Profesor.java → Subclase que hereda de Persona y agrega materia.
4. Main.java → Programa principal que instancia objetos y muestra información.
